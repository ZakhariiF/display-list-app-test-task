# Display movies app

## How run this app

To run this app you should fun command `npm i` and `npm run dev`

## Folder Structure

```javascript
|--src/
   |--app/
      |--components; // Custom components
      |--providers; // Providers
      |--api; // json mock data
      |--movie; // movie page
   |--layout.ts; // layout
   |--page.ts; // root page
```
